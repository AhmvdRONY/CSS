الفصل 1: كيفية استدعاء العناصر في CSS

قبل أي تعديل على التصميم لازم نفهم إزاي نوصل للعنصر صح. كل طريقة لها استخدام معين:

1) استدعاء العنصر Tag Selector

بنستخدمه لما نريد نعدل كل العناصر من نفس النوع. مثلاً كل <p> في الصفحة.

p { color: red; }

استخدامه: مناسب للعناصر العامة اللي مش محتاج تخصص.

2) استدعاء كلاس Class

الأكثر استخدامًا في CSS لأنه بيدي تحكم كامل بدون ما يأثر على باقي العناصر.

.box { padding: 20px; }

استخدامه: لما عايز تعمل ستايل لعنصر معين أو مجموعة محددة.

3) استدعاء ID

بيستخدم للعنصر اللي بيظهر مرة واحدة في الصفحة.

#main { background: black; }

استخدامه: للعناصر الفريدة زي الهيدر — ومينصحش بالاعتماد عليه للستايلات القابلة لإعادة الاستخدام.

4) استدعاء عنصر داخل عنصر

بنستخدمه لما عايزين نعدل عنصر موجود داخل مكان معين.

div p { font-size: 18px; }

استخدامه: لما نفس العنصر بيتكرر لكن عايز تعدل نسخة موجودة في منطقة معينة.

5) Attribute / Pseudo

اختيارات بالسمات أو بالـpseudo مثل a[href$=".pdf"] أو a:hover.

الفصل 2: الخلفيات Background

الخلفيات تعتبر من أقوى عناصر التصميم. كل خاصية لها قيمة معينة وكل قيمة لها دور محدد.

background-color

وظيفتها: تحديد لون الخلفية فقط.

متى نستخدمها؟ لعناصر بسيطة أو لو أردت تحسين الأداء.

background-color: #222;

background-image

وظيفتها: وضع صورة أو تدرج كخلفية.

  • url('img.jpg') — صورة.
  • linear-gradient() — تدرج لوني يمكن الاستغناء به عن صور ثقيلة.
  • radial-gradient() — تدرج مركزي.
  • none — لإلغاء الخلفية التصويرية.
background-image: url("img.jpg");

background-size

وظيفتها: تحدد كيفية ملء الصورة للحاوية.

  • cover — تغطي العنصر بالكامل (مثالي للـhero).
  • contain — تظهر الصورة كاملة بدون قص.
  • auto — الحجم الطبيعي للصورة.
  • 100% 100% — يعيد تحجيم الصورة وقد يشوهها.
background-size: cover;

background-repeat

وظيفتها: ضبط تكرار الصورة.

  • repeat
  • no-repeat
  • repeat-x
  • repeat-y

background-position

وظيفتها: مكان ظهور الصورة داخل العنصر.

  • center
  • top, bottom, left, right
  • 50% 20% — إحداثيات نسبية.
background-position: center;

نصيحة مهمة

ضع overlay (تراكب لوني) عندما تستخدم صورًا خلفية ويوجد نص فوقها لضمان تباين مقروئية جيد.

الفصل 3: البادينج Padding

وظيفته: هو المسافة الداخلية داخل العنصر، بين المحتوى والحد.

القيم:

  • padding-top
  • padding-bottom
  • padding-left
  • padding-right
  • padding: top right bottom left; — اختصار.
padding: 20px;

متى تستخدمه: داخل الأزرار والبطاقات لإعطاء نفس المسافة حول المحتوى. استخدم em/rem للنسب المعتمدة على حجم الخط.

الفصل 4: المارجن Margin

وظيفته: مسافة خارجية بين العنصر والعناصر المحيطة به.

قواعد ونصائح:

  • يمكن أن يحدث margin collapsing بين عناصر البلوك — راعِ ذلك.
  • استخدم margin: 0 auto; لمركزة عرض ثابت.
margin: 20px;

الفصل 5: الحدود Border

الحدود تُستخدم لتحديد إطار حول العنصر.

border-width

border-width: 2px;

border-style

  • solid
  • dashed
  • dotted
  • double

border-color

border-color: #fff;

border-radius

border-radius: 10px;

الفصل 6: النصوص Text

تحكم شامل في مظهر النصوص.

color

color: #fff;

font-size

font-size: 20px;

font-weight

  • normal / bold / 100-900

text-align

  • right / left / center / justify

line-height

line-height: 1.8;

الفصل 7: الألوان Colors

  • HEX — مثال: #ff00aa
  • RGB — مثال: rgb(255,0,0)
  • RGBA — يضيف شفافية: rgba(0,0,0,0.6)
  • HSL — تحكم أدق بالدرجة/الإشباع/السطوع.
color: rgba(255,255,255,0.8);

الفصل 8: العرض والارتفاع Width & Height

  • width وheight
  • min-width / max-width / min-height / max-height
  • fit-content أو auto للحالات المرنة
width: 100%; max-width: 1200px;

الفصل 9: العرض Display

تحدد طريقة ظهور العنصر وتفاعله مع العناصر الأخرى.

block

يكسر السطر ويأخذ العرض الكامل.

inline

يكون ضمن السطر ولا يقبل width/height.

inline-block

يجمع بين السلوكين.

flex / grid

للتصميمات المتقدمة.

الفصل 10: Flexbox

نظام مرن لترتيب العناصر في بعد واحد.

خواص أساسية

  • display: flex;
  • flex-direction: row | column
  • justify-content: flex-start | center | space-between
  • align-items: stretch | center
  • gap للمسافات بين العناصر
.row{ display:flex; gap:16px; }

الفصل 11: Grid

نظام ممتاز لتصميم تخطيطات ثنائية الأبعاد.

display:grid; grid-template-columns: repeat(3,1fr); gap:20px;

الفصل 12: المواضع Position

  • static — القيمة الافتراضية.
  • relative — إزاحة بالنسبة لموقعه الطبيعي.
  • absolute — يخرج من التدفق ويحدد حسب أقرب حاوية position.
  • fixed — يثبت بالنسبة للنافذة.
  • sticky — يلتصق بعد عبور نقطة معينة عند التمرير.
position: absolute; top: 20px; right: 20px;

الفصل 13: الظلال Shadows

تحكم في ظل العناصر والنصوص.

box-shadow: 0 8px 30px rgba(0,0,0,0.5); text-shadow: 0 2px 6px rgba(0,0,0,0.6);

الفصل 14: Transitions — الانتقالات

ما هي: الانتقالات تسمح بتنعيم التغيير بين حالتين عند تعديل خاصية CSS (مثل hover).

الخـواص

  • transition-property
  • transition-duration
  • transition-timing-function
  • transition-delay
.card{ transition-property: transform, box-shadow; transition-duration: 0.25s; transition-timing-function: ease; } .card:hover{ transform: translateY(-6px); box-shadow: 0 12px 30px rgba(0,0,0,0.5); }

الفصل 15: Transform — التحويلات

ماذا تفعل: تغيير موضع/حجم/دوران العنصر بدون إخراجه من التدفق (GPU-accelerated).

أمثلة:

.btn{ transition: transform .25s; } .btn:hover{ transform: translateY(-4px) scale(1.03); }

الفصل 16: Animations — الرسوم المتحركة

بناء حركات معقدة باستخدام @keyframes و animation.


@keyframes float { 0%{ transform: translateY(0) } 100%{ transform: translateY(-8px) } }
.icon{ animation: float 3s ease-in-out infinite alternate; }

الفصل 17: Filters & Blend Modes

filter يطبق تعديلات بصرية: blur(), brightness(), contrast(), saturate()

mix-blend-mode يتحكم باندماج الألوان بين طبقات العناصر.

img.hero{ filter: saturate(1.2) contrast(1.05); }

الفصل 18: Opacity — الشفافية

قيمة: من 0 إلى 1 — تؤثر على العنصر بكامله.

opacity: 0.6;

لو تريد خلفية شفافة فقط استخدم rgba على background بدلاً من opacity للعناصر الروتينية.

الفصل 19: Overflow — التحكم في التدفق

  • visible
  • hidden
  • scroll
  • auto
.box{ overflow: auto; }

الفصل 20: Cursor — شكل المؤشر

تحكم بسيط لكن مهم لتجربة المستخدم.

cursor: pointer;

قِيَم شائعة: default, pointer, move, text, wait, crosshair.

الفصل 21: Z-index & Stacking Context

z-index يحدد ترتيب طبقات العناصر عند التداخل — يجب أن يكون العنصر لديه position (غير static).

.modal{ position: fixed; z-index: 9999; }

احترس من stacking contexts الناتجة عن خصائص كـ transform أو opacity.

الفصل 22: Media Queries — التصميم المتجاوب

أمثلة سريعة للـ breakpoints:

@media (max-width: 768px) {
  .section{ padding: 12px; }
  header{ padding: 18px; }
}

أفضل مسار: mobile-first — اكتب قواعد الموبايل ثم أضف قواعد الشاشات الأكبر.

الفصل 23: Variables & calc()

متغيرات CSS لتوحيد الألوان والمسافات:

:root{ --accent: #00aaff; --gap: 16px; }
.btn{ padding: var(--gap); background: var(--accent); }
width: calc(100% - 40px);

الفصل 24: Object-fit & Object-position

مفيد جدًا لعرض الصور بطريقة متناسقة داخل البطاقات.

img{ object-fit: cover; object-position: center; }

الفصل 25: Mix-blend-mode & Backdrop-filter

mix-blend-mode للتحكم في تراكب الألوان.

backdrop-filter مفيد لتأثيرات الزجاج (glass effect).

.glass{ backdrop-filter: blur(12px); -webkit-backdrop-filter: blur(12px); }
