دورة CSS – شرح تفصيلي كامل

الفصل 1: كيفية استدعاء العناصر في CSS

قبل أي تعديل على التصميم لازم نفهم إزاي نوصل للعنصر صح. كل طريقة لها استخدام معين:

1) استدعاء العنصر Tag Selector

بنستخدمه لما نريد نعدل كل العناصر من نفس النوع. مثلاً كل <p> في الصفحة.

p { color: red; }

استخدامه: مناسب للعناصر العامة اللي مش محتاج تخصص.

2) استدعاء كلاس Class

الأكثر استخدامًا في CSS لأنه بيدي تحكم كامل بدون ما يأثر على باقي العناصر.

.box { padding: 20px; }

استخدامه: لما عايز تعمل ستايل لعنصر معين أو مجموعة محددة.

3) استدعاء ID

بيستخدم للعنصر اللي بيظهر مرة واحدة في الصفحة.

#main { background: black; }

استخدامه: للعناصر الفريدة زي الهيدر — وميفضّلوش في التصميم الكتير.

4) استدعاء عنصر داخل عنصر

بنستخدمه لما عايزين نعدل عنصر موجود داخل مكان معين.

div p { font-size: 18px; }

استخدامه: لما نفس العنصر بيتكرر لكن عايز تعدل نسخة موجودة في منطقة معينة.

الفصل 2: الخلفيات Background

الخلفيات تعتبر من أقوى عناصر التصميم. كل خاصية لها قيمة معينة وكل قيمة لها دور محدد.

background-color

وظيفتها: تحديد لون الخلفية فقط.

متى نستخدمها؟ لما نريد نخلي العنصر واضح بنبرة لون ثابتة.

background-color: #222;

background-image

وظيفتها: وضع صورة خلفية.

القيم المتاحة:

  • url(): لعرض صورة ثابتة.
  • linear-gradient(): لعمل تدرج لوني.
  • none: إلغاء الصورة.

استخدامها: لما نريد خلفية جذّابة أو تأثير بصري.

background-image: url("img.jpg");

background-size

وظيفتها: تحدد حجم صورة الخلفية.

القيم:

  • cover: تغطي العنصر بالكامل — الأفضل للصفحات.
  • contain: تظهر الصورة كاملة بدون تقطيع.
  • auto: الحجم الطبيعي.
  • 100% 100%: تمدد كامل لكن ممكن يشوه الصورة.
background-size: cover;

background-repeat

وظيفتها: تكرار أو منع تكرار الخلفية.

  • repeat: الصورة تتكرر في كل اتجاه.
  • no-repeat: تظهر مرة واحدة فقط.
  • repeat-x: تتكرر أفقي.
  • repeat-y: تتكرر عمودي.

background-position

وظيفتها: تحدد مكان ظهور الخلفية.

  • center: في المنتصف.
  • top / bottom / left / right: زاوية محددة.
  • 50% 20%: تحديد نسبة مخصصة.
background-position: center;

الفصل 3: البادينج Padding

وظيفته: هو المسافة الداخلية داخل العنصر، بين المحتوى والحد.

القيم:

  • padding-top: المسافة من الأعلى.
  • padding-bottom: المسافة من الأسفل.
  • padding-left: من اليسار.
  • padding-right: من اليمين.

متى نستخدمه؟ عندما يكون العنصر مكتوم أو الكلمات لاصقة في الحواف.

padding: 20px;
padding: 10px 20px;
padding-top: 40px;

الفصل 4: المارجن Margin

وظيفته: مسافة خارجية بين العنصر والعناصر المحيطة به.

القيم:

  • margin-top
  • margin-bottom
  • margin-left
  • margin-right

متى نستخدمه؟ للفصل بين العناصر — خاصة في التصميمات النظيفة.

margin: 20px;

الفصل 5: الحدود Border

الحدود تُستخدم لتحديد إطار حول العنصر، ولكل قيمة دور مختلف.

border-width

تحدد سُمك الحدود.

border-width: 2px;

استخدامه: إبراز العنصر أو تحديده.

border-style

شكل الحدود.

  • solid: خط عادي.
  • dashed: شرطات.
  • dotted: نقط.
  • double: خط مزدوج.
  • none: بدون حدود.

border-color

لون الحدود.

border-color: #fff;

border-radius

يجعل الحواف دائرية.

border-radius: 10px;

استخدامه: لإعطاء شكل ناعم وحديث.

الفصل 6: النصوص Text

يمثل تحكم شامل للمحتوى المكتوب في صفحة الويب.

color

font-size

حجم الخط.

font-size: 20px;

font-weight

سُمك الخط.

  • normal
  • bold
  • 100 - 900

text-align

محاذاة النص.

  • right
  • left
  • center
  • justify

line-height

المسافة بين السطور.

line-height: 1.8;

الفصل 9: العرض Display

تحدد طريقة ظهور العنصر وتفاعله مع بقية العناصر.

block

العنصر يأخذ الصف بالكامل.

inline

العنصر لا يكسر السطر.

inline-block

لا يكسر السطر + يقبل width و height.

flex

يشغل عنصر بنظام Flexbox.

grid

يشغل نظام Grid.

الفصل 10: Flexbox

أحد أهم أنظمة التوزيع الحديثة في CSS.

display: flex

تشغيل الفليكس.

flex-direction

اتجاه العناصر.

  • row (افتراضي)
  • column

justify-content

محاذاة أفقية.

  • center
  • space-between
  • flex-start
  • flex-end

align-items

محاذاة عمودية.

  • center
  • flex-start
  • flex-end

gap

المسافة بين العناصر.
